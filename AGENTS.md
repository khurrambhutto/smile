## Smile (Camera App)
In this project i am going to make a tauri based camera for linux, particularly ubuntu.
The app UI will be react and typescript which looks like native gnome app.

Use a native Rust camera backend on Linux via V4L2 for camera access; do not rely on browser `getUserMedia()` inside the desktop webview.
Optimize for a lightweight, production-ready Ubuntu build that should also work across major Linux distros with V4L2 support.
